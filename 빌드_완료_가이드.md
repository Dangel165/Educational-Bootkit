## 🚀 지금 바로 빌드하기

### 방법 1: 가장 빠른 방법 (추천)

```batch
QUICK_BUILD.bat
```

이 방법은:
- WDK 설치 불필요
- BootkitTools.exe 빌드
- BootkitTests.exe 빌드
- 약 1-2분 소요

### 방법 2: 설정 확인 후 빌드

```batch
# 1단계: 설정 검증
verify_setup.bat

# 2단계: 빌드
build_usermode.bat
```

### 방법 3: Visual Studio에서 빌드

1. `BootkitAnalysisFramework.sln` 더블클릭
2. Visual Studio에서 열림
3. 솔루션 탐색기에서 프로젝트 선택:
   - BootkitTools (우클릭 > 빌드)
   - BootkitTests (우클릭 > 빌드)
4. 또는 Ctrl+Shift+B (전체 빌드)

## 🔧 드라이버 빌드 (선택사항)

드라이버를 빌드하려면 WDK가 필요합니다.

### WDK 설치
1. Visual Studio Installer 실행
2. "개별 구성 요소" 탭
3. "Windows Driver Kit" 검색 및 설치
4. 또는 https://docs.microsoft.com/en-us/windows-hardware/drivers/download-the-wdk

### 드라이버 빌드
```batch
build.bat
```

### 드라이버 설치 (관리자 권한 필요)
```batch
# 1. 테스트 서명 활성화
scripts\enable_test_signing.bat

# 2. 재부팅

# 3. 드라이버 설치
sc create BootkitDriver type= kernel binPath= "C:\full\path\to\BootkitDriver.sys"
sc start BootkitDriver

# 4. 드라이버 상태 확인
sc query BootkitDriver
```

## ❓ 문제 해결

### "Visual Studio not found" 오류
```
해결: Visual Studio 2019 이상 설치
다운로드: https://visualstudio.microsoft.com/
```

### "C++ workload not found" 오류
```
해결:
1. Visual Studio Installer 실행
2. "수정" 클릭
3. "C++를 사용한 데스크톱 개발" 체크
4. "수정" 클릭하여 설치
```

### "MSBuild not found" 오류
```
해결: Visual Studio 재설치 또는 복구
```

### 빌드는 성공했지만 실행 안됨
```
확인사항:
1. bin\x64\Release 폴더에 .exe 파일 있는지 확인
2. 관리자 권한으로 실행 시도
3. Windows Defender 예외 추가
```

### WDK 관련 오류
```
해결 1: WDK 설치
해결 2: build_usermode.bat 사용 (드라이버 제외)
```
