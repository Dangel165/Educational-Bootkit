## ⚠️ 주의사항

### WDK 관련
- 드라이버 빌드는 WDK 필요
- WDK 없이도 사용자 모드 컴포넌트 빌드 가능
- WDK 설치: https://docs.microsoft.com/en-us/windows-hardware/drivers/download-the-wdk

### Visual Studio 버전
- Visual Studio 2019 이상 필요
- C++ 데스크톱 개발 워크로드 필수
- Windows SDK 10 권장

### 관리자 권한
- 드라이버 설치 시 관리자 권한 필요
- 빌드 자체는 일반 권한으로 가능
